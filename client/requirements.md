## Packages
framer-motion | Page transitions and smooth micro-interactions
recharts | Admin dashboard charts and statistics visualization
date-fns | Date formatting for enrollment timelines
react-dropzone | Better file upload handling for documents

## Notes
Authentication uses /api/auth/me to check current session
Role-based routing assumes user.role is 'student' or 'admin'
Document uploads expect base64 string in the payload
OCR data is mock-generated by backend and placed in document.ocrData
